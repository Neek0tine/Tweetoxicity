What is Tweetoxicity ?

Tweetoxicity is a program that analyses Twitter user behavior through their actions on their Twitter Profile. The program utilizes machine learning to give Twitter users appropriate score according to their tweets or retweets. This program is meant for educational purposes and no ill intentions exist prior to creating this program.

Sentiment Analysis

Using Sentiment Analysis, an NLP technique, Tweetoxicity can analyze whether the user has a high behavior score or not from the user's tweets. Results use different tests to find the most confident score. This does not indicate whether the user is bad or not, but instead provides an alaysis on their behavior on Twitter.

Twitter API

The program also utilizes Twitter API to scrape data from Twitter officially. In case of API failure or Authorization failures, Requests module will be used to manually scrape the data from Twitter, of course at a price of reduced request amount and increased time interval as to not be intrusive to Twitter. The scraped data is then sent to the program which keeps the machine learning model, and asks the program politely to assess the given user with its already made model through extensive training.
